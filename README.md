# PTZ Controller
PTZ Controller using SimplTrack/HuddleView HTTP-CGI comands

This is contains a simple set of pan-tilt and zoom/focus controls. Speed is fixed. Just set your camera IP address in index.html

If you need more presets, you can copy-paste. Just update the data-preset value for additional presets. This assumes you have 
access to HuddleView app to configure the preset locations.
